# 电商实时数仓项目（Flink 实时计算）

## 一、项目介绍
本项目是基于 Flink 的电商实时数据仓库解决方案，针对用户行为日志与业务数据，进行实时采集、清洗、聚合、计算与指标展示。
项目严格按照企业标准分层：ODS → DWD → DWM → DWS → ADS，实现高吞吐、低延迟、数据统一的实时计算链路。

## 二、使用技术
- 计算引擎：Flink
- 消息队列：Kafka
- 维度存储：HBase
- 实时计算：Flink SQL + DataStream
- 数据可视化：Doris 报表展示

## 三、项目架构
1. ODS 层：实时采集用户行为日志与业务数据
2. DWD 层：数据清洗、脱敏、规整、过滤脏数据
3. DWM 层：轻度聚合，中间结果计算
4. DWS 层：按主题宽表聚合
5. ADS 层：产出最终实时指标，用于报表展示

## 四、核心功能
- 用户行为实时分析
- 实时 UV / PV 计算
- 实时热门商品统计
- 实时流量来源分析
- 实时交易数据汇总

## 五、个人职责
- 负责实时数仓 DWD、DWS 层数据清洗与宽表构建
- 使用 Flink 实现实时数据分流、聚合、去重
- 维表关联、状态管理、窗口计算开发
- 实时指标开发与数据校验
- 保证实时任务稳定运行

## 六、项目亮点
- 采用事件时间 + 水位线，保证乱序数据准确计算
- 使用 Flink 状态 + TTL，防止状态膨胀
- 维度数据预加载，提升关联效率
- 全流程实时处理，数据延迟控制在秒级
